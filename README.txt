📱 Chess AI WebApp to APK wrapper using Capacitor

✅ WebApp URL: https://basic-chess-ai-min-kyu.vercel.app/

To use on Codemagic:
1. Upload this project to your GitHub repo
2. In Codemagic, connect to GitHub and select this repo
3. Set build platform: Android (Capacitor)
4. Click 'Start new build'
5. After build, download the APK
